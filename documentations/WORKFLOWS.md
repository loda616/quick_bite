# QuickBite Application Workflows

This document provides an overview of the main user workflows in the QuickBite application. Each workflow is described in detail in a separate file in the `workflows` directory.

## Main Workflows

1.  **[Authentication](./workflows/authentication_workflow.md)**: User registration, login, and password reset.
2.  **[Menu Browsing](./workflows/menu_browsing_workflow.md)**: Viewing categories and food items.
3.  **[Search](./workflows/search_workflow.md)**: Searching for food items.
4.  **[Cart Management](./workflows/cart_management_workflow.md)**: Adding, updating, and removing items from the cart.
5.  **[Checkout](./workflows/checkout_workflow.md)**: Placing an order and making a payment.
6.  **[Order Management](./workflows/order_management_workflow.md)**: Viewing past orders.
7.  **[Profile Management](./workflows/profile_management_workflow.md)**: Viewing and updating user profile information.
8.  **[Favorites Management](./workflows/favorites_management_workflow.md)**: Managing favorite food items.
